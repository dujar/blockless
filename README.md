# Decentralized Swap Interface

A Progressive Web App (PWA) that loads a swap interface with pre-filled information from URL parameters.

## Features

- Parses URL parameters for blockchain, amount, and target address
- Displays swap details in a clean interface
- Loads external swap interface in an iframe
- Works as a standalone PWA

## URL Parameters

The application accepts the following URL parameters:

- `blockchain`: The target blockchain for the swap
- `amount`: The amount to swap
- `targetAddress`: The recipient address

Example URL:
```
http://localhost:5173/?blockchain=ethereum&amount=1.5&targetAddress=0x1234567890123456789012345678901234567890
```

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## PWA Functionality

This application is a Progressive Web App that can be installed on mobile and desktop devices. It works offline and can be added to your home screen.
